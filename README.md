# Cadastro de Livros

Projeto simples em Java para praticar conceitos básicos de Programação Orientada a Objetos (POO).

## Funcionalidades

- Criar um objeto da classe `Livro`
- Armazenar título, autor e número de páginas
- Exibir um resumo do livro no console

## Exemplo de Saída

```text
Título: O Guia do Mochileiro das Galáxias
Autor: Douglas Adams
Páginas: 208
```

## Tecnologias

- Java
